# AI 划词翻译 — 图标设计需求

## 项目简介

**AI 划词翻译** 是一款 Chrome 浏览器扩展，允许用户在网页上选中文本后，通过大语言模型（LLM）进行翻译。支持多个 AI 提供商（DeepSeek、阿里云百炼、OpenAI 兼容接口），并内置了预设配置切换、主题切换等功能。

---

## 一、扩展图标（PNG 文件）

插件需要一套尺寸递进的 PNG 图标，用于浏览器工具栏和应用商店展示。

### 规格清单

| 尺寸 | 用途 | 当前状态 |
|---|---|---|
| **16 × 16 px** | 页面 favicon 上下文、扩展管理列表小图 | ✅ 已有（可替换） |
| **32 × 32 px** | 高 DPI 显示器工具栏按钮、Windows 任务栏 | ❌ 缺失，建议补充 |
| **48 × 48 px** | `chrome://extensions` 扩展管理页面列表 | ✅ 已有（可替换） |
| **128 × 128 px** | Chrome 网上应用店展示、扩展详情页主图 | ✅ 已有（可替换） |

### 目录结构

```
icons/
├── icon16.png       # 16×16
├── icon32.png       # 32×32  (建议新增)
├── icon48.png       # 48×48
└── icon128.png      # 128×128
```

### 技术要求

- **PNG 格式**（Chrome 扩展不支持 SVG 图标）
- **透明背景**（适配浅色/深色工具栏）
- 非交错（interlace 关闭）
- 16×16 下必须清晰可辨
- 避免过细线条（< 2px）、复杂渐变
- 内容保留 ~2px 安全边距（Chrome 自动施加圆角遮罩）

## 四、设计方向建议

### 主题关键词

翻译、AI、语言、地球、对话气泡、字母 A / T

### 方向一：🌐 + 文字（推荐）

- 地球/语言符号为主体
- 结合字母 **T**（Translation）或 **A**（AI）
- 示例思路：地球弧线上叠加一个翻译符号或字母

**优点：** 语义明确，小尺寸易识别

### 方向二：对话气泡 + 智能元素

- 聊天气泡内嵌星星 ✦ 或字母 AI
- 表示 AI 驱动的翻译

**优点：** 现代感强，符合 AI 主题
**缺点：** 16px 下气泡细节可能糊

### 方向三：极简字母

- 仅一个精心设计的字母 **T** / **A**
- 圆角、渐变、与齿轮呼应的线条风格

**优点：** 小尺寸极清晰
**缺点：** 首次见不容易猜出是翻译工具

### 方向四：齿轮 + 翻译符号（与悬浮按钮呼应）

- 齿轮轮廓内嵌翻译符号
- 与页面右下角 ⚙️ 悬浮按钮形成统一语言

**优点：** 两个入口保持一致体验

## 五、色彩与视觉规范

| 属性 | 值 |
|---|---|
| 主色 | `#4A90D9`（蓝色，与弹窗 header 一致） |
| 辅色 | `#FFFFFF` / `#E0E0E0` |
| 文字色 | `#333333` |
| 暗色背景 | `#2D2D2D` |
| 圆角风格 | 6px–12px |
| 阴影 | 柔和（`rgba(0,0,0,0.12)` ~ `0.18`） |
| 线条风格 | `stroke-width="2"`、`fill="none"` |

### 内联 SVG 技术约束

- 所有 SVG 使用 `stroke="currentColor"`，颜色由 CSS 控制
- `stroke-width="2"`，`fill="none"`
- viewBox 统一为 `0 0 24 24`

## 七、交付后需要在 manifest.json 中补充

```diff
{
  "icons": {
    "16": "icons/icon16.png",
    "48": "icons/icon48.png",
+   "32": "icons/icon32.png",
    "128": "icons/icon128.png"
  }
}
```

> 注意：`manifest.json` 中 `icons` 字段官方只保证 16/48/128 三档，32 是扩展额外支持的尺寸。即使不写入 manifest，把文件放在 `icons/` 目录下浏览器也会自动检测使用。
